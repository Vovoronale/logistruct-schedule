# Зовнішня маршрутизація стрілок залежностей

## Мета

Стрілки залежностей не повинні перетинати два блоки робіт, які вони з’єднують. Інші блоки на шляху стрілки не вважаються перешкодами.

## Геометрія

Для кожного видимого блока вимірюються ліва, права, верхня та нижня межі, а також вертикальний центр. Повна стрілка завжди виходить із середини правого краю блока-попередника та входить у середину лівого краю залежного блока.

Маршрут складається з п’яти ортогональних сегментів `H → V → H → V → H`:

1. горизонтальний вихід праворуч за межі попередника;
2. вертикальний перехід до коридору між рядками;
3. горизонтальний рух коридором до точки ліворуч від залежного блока;
4. вертикальний підхід до центра залежного блока;
5. горизонтальний вхід зліва до його межі.

Якщо залежний блок розташований нижче, коридор проходить між нижньою межею попередника та верхньою межею залежного блока. Якщо залежний блок розташований вище, використовується проміжок між нижньою межею залежного блока та верхньою межею попередника. Координата коридору обмежується цим проміжком, тому перший і останній вертикальні сегменти не заходять усередину пов’язаних блоків.

Горизонтальні зовнішні точки мають сталий мінімальний відступ від країв блоків. Для паралельних ребер застосовується невелике детерміноване зміщення всередині доступного коридору; якщо місця мало, зміщення стискається, а не виштовхує маршрут у блок.

## Межі зміни

Змінюється лише маршрутизація повних зв’язків із двома видимими кінцями. Відбір ребер, кольори, наконечники, SVG-overlay та короткі позначки для прихованих робіт залишаються без змін.

## Перевірка

Модульні тести перевіряють:

- маршрут `H → V → H → V → H` без діагоналей;
- вихідну вертикаль праворуч від попередника;
- вхідну вертикаль ліворуч від залежного блока;
- коридор між межами рядків для напрямків угору та вниз;
- розведення паралельних ребер без виходу коридору в блок;
- незмінну поведінку позначок прихованих зв’язків.

Компонентний тест підтверджує вимірювання верхньої та нижньої межі блоків і передавання цих координат маршрутизатору.
